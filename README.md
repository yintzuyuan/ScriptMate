# ScriptMate

![Version](https://img.shields.io/badge/Version-0.9.0--alpha.1-blue)
![Platform](https://img.shields.io/badge/Platform-macOS%2014.0%2B-lightgrey)
![Glyphs](https://img.shields.io/badge/Glyphs-3-yellow)

**AI Scripting Workbench for Glyphs 3** — Generate, refine, and execute Python scripts through conversational AI.

## Features

- **Three-panel architecture** — Chat, Code, and Output panels working together
- **Three conversation modes** (`Shift+Tab` to switch):
  - **Chat** — Q&A about Glyphs API, no code execution
  - **Console** — One-shot script generation with auto-execution
  - **Window** — GUI tool builder with Vanilla UI
- **BYOK (Bring Your Own Key)** — Works with any OpenAI-compatible AI service
- **Glyphs API live search** — AI queries Python API docs and mekkablue script library
- **Script import/export** — Save scripts to Glyphs Script menu or load existing `.py` files
- **Localization** — English, Traditional Chinese, Simplified Chinese

## Requirements

- macOS 14.0 or later
- [Glyphs 3](https://glyphsapp.com)

## Installation

1. Download `ScriptMate.glyphsPlugin.zip` from the [latest release](https://github.com/yintzuyuan/ScriptMate/releases/latest)
2. Unzip and double-click `ScriptMate.glyphsPlugin` to install
3. Restart Glyphs

## Setup

ScriptMate requires an API key from an OpenAI-compatible AI provider. Open **Window > ScriptMate**, then click the settings icon to configure your provider and API key.
